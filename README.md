### Prerequisites
- [Kotlin](https://kotlinlang.org/) (included with the project)
- JDK 17 or higher

### Running a specific day
```bash
# Run Day 01
./gradlew run --args="Day01"

# Or run directly with Kotlin
cd src
kotlin Day01.kt
```

### Running all solutions
```bash
./gradlew run
```
